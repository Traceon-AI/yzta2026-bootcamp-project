# Sprint 2 — Daily Scrum Kayıtları

Format: her giriş *Dün / Bugün / Blocker* üçlüsüyle, 2-3 cümleyi geçmeden.

Kayıtlar WhatsApp/Slack'ten haftada en az 2 kez buraya aktarılır.

---

## 8 Temmuz

- **Eylül:** Dün: Doküman işleme mimarisi üzerinde çalıştım. Bugün PDF/DOCX okuma yapısını modüler hale getirmeyi planlıyorum. Blocker: Yok.
- **Önder:** Dün: Frontend ekranlarını gözden geçirdim. Bugün eksik UX notlarını çıkaracağım. Blocker: Backend API henüz hazır değil.
- **Efnan:** Dün: Entegrasyon planını inceledim. Bugün Docker ve API bağlantısı için hazırlık yapacağım. Blocker: Backend endpointlerinin netleşmesini bekliyorum.
- **Ezgi:** Dün: GDPR/KVKK veri setlerini incelemeye başladım. Bugün test dokümanı hazırlığına devam edeceğim. Blocker: Yok.
- **Ahmet Faruk:** Dün: Sprint planı ve repo düzenini gözden geçirdim. Bugün backend mimarisi ve AI analiz akışı üzerine çalışacağım. Blocker: LLM tasarımının netleşmesi gerekiyor.

## 12 Temmuz

- **Eylül:** Dün: Doküman işleme ve chunk yapısı üzerinde ilerledim. Bugün API sözleşmesine uygun çıktıyı oluşturmaya odaklanacağım. Blocker: Yok.
- **Önder:** Dün: Frontend tarafında mock veri ile testlere devam ettim. Bugün backend entegrasyonu için hazırlık yapacağım. Blocker: Gerçek API henüz hazır değil.
- **Efnan:** Dün: Docker görevleri için araştırma yaptım. Bugün entegrasyon hazırlıklarını sürdüreceğim. Blocker: Backend bağlantısı bekleniyor.
- **Ezgi:** Dün: GDPR maddelerini kontrol ettim. Bugün test senaryolarını geliştireceğim. Blocker: Yok.
- **Ahmet Faruk:** Dün: Sprint ilerleyişini değerlendirdim. Bugün eksik kalan görevleri yeniden planlayacağım. Blocker: Sprint içinde görünür ilerleme düşük.

---

## 16 Temmuz

- **Eylül:** Dün: Backend tarafında gerçek doküman işleme akışını tamamladım. Bugün dinamik regülasyon okuma ve çıktı formatını sonlandıracağım. Blocker: Gemini ücretsiz kullanım limitleri.
- **Önder:** Dün: Frontend görevlerini gözden geçirdim. Bugün Sprint 3 için planlanan UX geliştirmelerini hazırlayacağım. Blocker: API bağlantısı.
- **Efnan:** Dün: Entegrasyon görevlerini planladım. Bugün backend hazır olduğunda kullanılacak akışı netleştireceğim. Blocker: Gerçek endpointlerin tamamlanması.
- **Ezgi:** Dün: KVKK test dokümanı hazırlıklarını sürdürdüm. Bugün veri kontrolüne devam edeceğim. Blocker: Yok.
- **Ahmet Faruk:** Dün: Repo düzeni ve Sprint planlamasını güncelledim. Bugün backend AI katmanını devralacak mimariyi planlayacağım. Blocker: Rate limit nedeniyle LLM analizinin yeniden tasarlanması gerekiyor.

---

## 19 Temmuz

- **Eylül:** Dün: Backend gerçek JSON çıktısı ve Swagger testlerini tamamladım. Bugün teknik devir dokümanını hazırlayıp teslim edeceğim. Blocker: LLM analizinin batch mimariye geçirilmesi gerekiyor.
- **Önder:** Dün: Sprint 3 görev planını güncelledim. Bugün frontend entegrasyonu için hazırlıkları tamamlayacağım. Blocker: Yok.
- **Efnan:** Dün: Sprint 3 entegrasyon planını netleştirdim. Bugün Docker ve deployment görevlerine hazırlanacağım. Blocker: Yok.
- **Ezgi:** Dün: Sprint 3 test görevlerini devraldım. Bugün test dokümanlarını planlayacağım. Blocker: Yok.
- **Ahmet Faruk:** Dün: Repo yapısını düzenledim, .env ve güvenli yapılandırmayı tamamladım. Bugün Sprint 3 için embedding + batch LLM mimarisini planlayıp görevleri yeniden dağıttım. Blocker: Gemini ücretsiz API hız limiti.

---

## Sprint Ortası Notu (12-13 Temmuz — Entegrasyon Testi)

Planlanan entegrasyon testi gerçekleştirilemedi. Backend geliştirme süreci beklenenden uzun sürdüğü için frontend ile gerçek API entegrasyonu yapılamadı. Sprint sonunda backend'in gerçek çıktı üretir hale gelmesiyle entegrasyon Sprint 3'e taşındı ve doğrudan gerçek API üzerinden yapılacak şekilde yeniden planlandı.

---

## Sprint Sonu Özeti (19 Temmuz)

Sprint boyunca üç yazılı daily kaydı oluşturuldu ve sprint sonunda kapsamlı durum değerlendirmesi yapıldı. İletişim özellikle sprint sonuna doğru yoğunlaştı; backend tarafındaki büyük geliştirmeler sprint kapanışında repoya aktarıldı. En önemli blocker ücretsiz Gemini API hız limiti ve backend çalışmalarının ara push'larla görünür olmamasıydı. Sprint sonunda yeniden planlama yapılarak Sprint 3 hedefleri netleştirildi.
